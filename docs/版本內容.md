### 新增：育寵系統

#### 系統定位
- 新增「育寵師」身份，核心定位為生活生態型玩法，兼具戰鬥輔助功能
- 官方說明：寵物本身**不附帶任何屬性養成**，不直接提升玩家面板數值（攻擊/破防/會心/元素等）

#### 主要玩法
- 捕捉野外萌寵
- 寵物繁育（可遺傳外觀、性格、特質）
- 寵物打工（莊園種植、挖礦、捕魚等生產活動）
- 開放世界探索輔助（御寒、照明、謎題解鎖等）
- 寵物對戰（閒趣玩法）

#### 特質系統

特質是寵物的核心能力詞條，性質分兩種：
- **大多數特質**：被動生效，持有即有效果（莊園、採集、探索等）
- **戰鬥類特質**：類似主動技能，需手動拖入**角色技能欄**才在戰鬥中生效

每隻寵物最多攜帶 **4 個特質**：1 個戰鬥類（主動）+ 最多 3 個被動（同類型只能帶 1 個）

| 特質類型 | 說明 |
|----------|------|
| 戰鬥類 | 輔助玩家戰鬥，需佔用角色技能欄 |
| 莊園類 | 自動種植、灌溉、收穫、畜牧 |
| 探索類 | 自動解謎、偵測寶箱、提供照明 |
| 採集類 | 自動挖礦、採草藥、釣魚 |
| 閒趣類 | 演奏、騎乘、趣味動作等 |

- 特質品質分四級：初 → 優 → 精 → 絕
- 提升親密度（上限 8 級）可隨機獲得新特質或升等
- 特質品質精品與絕品「性能差異較小」，絕品主要是數值微幅提升
- 萌爪商店：消耗 **15 萌爪幣·稀** 可直接兌換絕品寵物技能特質書（無數量限制）
- 萌爪谷扭蛋機：消耗萌爪幣·稀，也可抽到技能書
- 同種類異性寵物達親密度 3 級後可繁育，後代可遺傳父母的特質、性格、外觀

**萌爪幣·稀 取得方式：**

| 來源 | 數量 |
|------|------|
| 每週前 10 次成功捕捉野生動物 | 各 1 個（共最多 10 個/週） |
| 升級寵物圖鑑（2→3 級時） | 5 個 |
| 珍萌大會活動 | 不定量 |

#### 寵物性格
- 性格影響戰鬥技能表現、莊園生產效率、陪伴互動風格
- **技能書本身附帶特定性格加成**，需讓寵物性格與技能書對應才能發揮加成
- 性格不符可用「萌寵性格百變丸」調整（圖鑑 2 級 + 身份等級 3 級後可自製，或在交易行購買）

#### 解鎖條件
- 角色 60 級 + 完成主線任務「尋覓寵物」，解鎖育寵師身份

#### 育寵師身份等級
- 身份等級上限 15 級（2025 賽季起提升至 20 級）
- 每週五、日晚 8 時舉辦「珍萌大會」可獲稀有萌寵

#### 珍萌大會
- **時間**：每週五、週日晚上 8 點，於「仙居原」地圖舉辦
- **玩法**：限時出現平時不會在野外刷新的**稀有寵物品種**（珍萌王、小珍萌等）
- **所需工具**：「珍萌葫蘆」（在集市購買材料後，於身份 → 育寵 → 技藝製作）
- **獎勵**：萌爪幣·稀、珍萌王腳印、小珍萌腳印，可兌換稀有寵物與外觀

#### 如何捕捉野生寵物

**步驟：**
1. 身份 → 育寵 → 技藝，製作「捉宠葫」
2. 前往大世界，鎖定野生寵物
3. 點「嘗試捕捉」→ 長按聚焦 → 成功收入囊中

**提高成功率的方式：**
- 升級寵物圖鑑（提升捕捉機率 + 擴大持有上限）
- 使用較高品質的捉宠葫
- 捕捉前對寵物「安撫」或「餵食」讓牠放鬆警戒
- 捕捉失敗**不消耗**捉宠葫，可安心嘗試

**查詢寵物出沒位置：** 地圖 → 標籤 → 身份 → 育寵

| 秒爪回春 | PVE | 治療 + 增益 | — |

#### 九靈 / 神相 職業推薦配置

| 面向 | 推薦 |
|------|------|
| 寵物外觀 | 優先稀有神獸（珍萌大會），外觀不影響職業適配 |
| 性格 | 依所選技能書標示的性格加成決定，不符合再用百變丸調整 |
| PVE 戰鬥特質 | **寵追猛打** + 解鎖「協力同心」被動 |
| PVP 戰鬥特質 | **萌卷殘雲** 或 **萌衝直撞** |
| PVE + PVP 兩用 | **人寵合璧** |

- **九靈**：寵物系統與九靈召喚物（女藥人/魂燈）是完全不同系統，不互相影響；寵追猛打可作為心鋒的替代輔助
- **神相**：自身高命中/高會心，寵追猛打每次攻擊傷害相對可觀，相容性佳

#### 對傷害計算器的影響
- 核心公式（防禦穿透率、元素穿透率、會心率、命中率）輸入數值**不受影響**，無需修改計算器
- 寵物戰鬥技能（如「寵追猛打」）為獨立額外傷害來源，類似心鋒脫手傷害，約佔總傷害 6–8%，建議在計算器外單獨估算
