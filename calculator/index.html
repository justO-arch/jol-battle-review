另存新檔（建立獨立存檔的複本）
本計算機的所有存檔（素質數值、BOSS 預設等）存在瀏覽器的 localStorage 中。 同一台電腦上的多個複本若使用相同識別碼，存檔會互相覆蓋。 若需要讓每個複本各自獨立儲存，請依下列步驟操作：
  1. 複製 calculator.html，改名為 calculator2.html（或任意名稱）
  2. 用文字編輯器（記事本即可）開啟該複本
  3. 找到這一行：const STORAGE_KEY = 'jol';
  4. 將 'jol' 改為不同的識別碼，例如 'jol2'、'jol3' 等
  5. 儲存檔案，之後兩個複本的存檔即完全獨立

公式說明（唯讀）
── Δ 定義 ──────────────────────────────────────
Δ防禦　= 敵方防禦力 − 我方破防　　　　Δ抗性　= 敵方元素抗性 − 我方忽視元素抗性
Δ會心　= 我方會心 − 敵方會心抗性　　　Δ命中　= 我方命中 − 敵方格擋
── 穿透 ────────────────────────────────────────
防禦穿透率　= A防 ÷ (Δ防禦 + A防) − B防
元素穿透率　= A元 ÷ (Δ抗性 + A元)
── 命中 / 會心 ──────────────────────────────────
會心率　　　= ( A會 × Δ會心 + B會 ) ÷ ( Δ會心 + C會 )
命中率　　　= A命 + ( B命 × Δ命中 ) ÷ ( Δ命中 + C命 )
── 傷害 ────────────────────────────────────────
氣盾衰減　　= 若 氣盾 ≥ 3×破盾：剩 氣盾−2×破盾；否則：剩 0.5×(氣盾−破盾)；下限 0
元素傷害　　= k₃ × 元素攻擊 × 元素穿透率
攻克傷害　　= k₁ + k₂ × (攻擊 − 剩餘氣盾 + 克制 − 抵禦) × 防禦穿透率
PVE 期望　　= (攻克 + 元素) × [ 命中率×(會心率×會心倍率 + (1−會心率)) + 0.5×(1−命中率) ] × 獨立乘區
PVP 期望　　= (攻克 + 元素) × [ 命中率×(會心率×(會心倍率−抗會傷) + (1−會心率)) + 0.5×(1−命中率) ] × 獨立乘區
